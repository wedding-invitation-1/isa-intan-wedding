# Folder Gambar / Images

## Struktur Folder

### /pengantin/
Foto pasangan pengantin (digunakan di bagian Mempelai & Cover)
- `isa.jpg` → Foto mempelai pria
- `intan.jpg` → Foto mempelai wanita

### /galeri/
Foto-foto galeri prewedding / momen bersama
- `galeri-1.jpg` s/d `galeri-7.jpg`
- Format yang disarankan: JPG/WEBP, ukuran max 800KB per foto

### /keluarga/
Foto avatar anggota keluarga besar (opsional)
- Bisa menggunakan emoji avatar atau foto asli
- Format: JPG/PNG, ukuran 200x200px
